# Модернізація та оптимізація інтернет-магазину «Епіцентр»

Цей репозиторій створено для командної розробки проєкту з оновлення платформи електронної комерції.

## Основні цілі:
* Оптимізація UX/UI та навігації.
* Синхронізація залишків у реальному часі.
* Підвищення масштабованості та відмовостійкості.

## Технологічний стек:
* **Frontend**: React.
* **Backend**: Node.js / Python.
* **Database**: PostgreSQL.
* **DevOps**: Docker.
